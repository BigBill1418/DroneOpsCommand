# v2.67.0 Mission Hub Redesign — Orchestration Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to dispatch the parallel agents below.

**Goal:** Ship v2.67.0 — replace the linear MissionNew/Edit wizard with a Mission Hub + Facet pattern; preserve every existing mission and every 2026-05-02/03 shipped feature; pass all 15 done-definition checkboxes from spec §11.

**Architecture:** 3 parallel aegis agents (A Hub / B Facets-1 / C Report-facet) in isolated worktrees + 1 sequential aegis agent (D Routing + legacy preservation + contract tests + ADR-0014) merging last. **Tech Stack:** Python 3.12 / FastAPI / SQLAlchemy 2 / asyncpg / React 19 / TypeScript / Mantine / Vite / Docker Compose / Stripe API / ntfy / Cloudflare Access / Guardian Graph mailer (for completion email).

**Spec:** `docs/superpowers/specs/2026-05-03-mission-hub-redesign-design.md`
